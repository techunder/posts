# ж•°еӯҰиЎЁиҫҫ

й—®йўҳжӢҶи§ЈеҰӮдёӢпјҡ

- **еҶізӯ–еҸҳйҮҸ**пјҡи®ҫй•ҝж–№еҪўйқ еўҷзҡ„дёҖиҫ№дёәй•ҝпјҢй•ҝдёә $x$ зұіпјҢе®Ҫдёә $y$ зұіпјҢеӣ дёәйқўз§Ҝдёә $xy$пјҢжүҖжұӮе°ұжҳҜ $argmax_{x,y}\left(xy\right)$

- **зӣ®ж ҮеҮҪж•°**пјҡжұӮйқўз§Ҝзҡ„жңҖеӨ§еҖјпјҲmaxпјүпјҢеҚі $max(xy)$

- **зәҰжқҹжқЎд»¶**пјҡ$x+2y=8$пјҢ$x \gt 0$пјҢ$y \gt 0$

> [!WARNING]
> е®Ңж•ҙзҡ„ж•°еӯҰиЎЁиҫҫдёәпјҡ$argmax_{x,y} \left(xy\right) \quad s.t. \quad x+2y=8, x>0, y>0$

> еӣ дёәзӣ®ж ҮеҮҪж•°дёә $xy$пјҢж¬Ўж•°й«ҳдәҺ1пјҢжүҖд»ҘжҳҜйқһзәҝжҖ§и§„еҲ’

# ж•°еӯҰжұӮи§Ј

```katex

```

д»Је…Ҙзӣ®ж ҮеҮҪж•°еҫ—

жұӮиҝҷдёӘеҮҪж•°зҡ„**еҜјж•°**е№¶и®©е…¶зӯүдәҺ 0 еҚіеҸҜи®Ўз®—еҮәжҠӣзү©зәҝзҡ„жңҖеҖјпјҲеӣ дёәејҖеҸЈеҗ‘дёӢпјҢжүҖд»ҘдёҖе®ҡжҳҜжңҖеӨ§еҖјпјү

$f^'(y)=-2 \times 2y+8=0$ еҫ— $y=2$

еҪ“ $y=2$ ж—¶пјҢ

$f(y) = -2y^2 + 8y = 8$

$x=8-2y=4$

д№ҹе°ұжҳҜеҪ“й•ҝж–№еҪўзҡ„й•ҝдёә 4 зұіпјҢе®Ҫдёә 2 зұіж—¶пјҢйёЎеңҲеҸҜиҫҫеҲ°жңҖеӨ§йқўз§Ҝ 8 е№іж–№зұігҖӮ

# зЁӢеәҸжұӮи§Ј

## ж–№жі•дёҖпјҡеҲ©з”Ё Scipy

`Scipy` жңүдёҖдёӘ [minimize](https://docs.scipy.org/doc/scipy/reference/generated/scipy.optimize.minimize.html) жЁЎеқ—пјҢеҸҜд»ҘеҫҲж–№дҫҝи®Ўз®—еҮәйқһзәҝжҖ§и§„еҲ’зҡ„жңҖдјҳи§Ј

```python
# pip install scipy
from scipy.optimize import minimize
import numpy as np

# жңҖе°ҸеҢ–зӣ®ж ҮеҮҪж•°пјҡжҲ‘д»¬иҰҒжңҖеӨ§еҢ– xyпјҢзӯүд»·дәҺжңҖе°ҸеҢ– -xy
def objective(x):
    return - (x[0] * x[1])  # x[0]=x, x[1]=y

# зӯүејҸзәҰжқҹпјҡx + 2y = 8
def constraint(x):
    return x[0] + 2 * x[1] - 8

# зәҰжқҹжқЎд»¶е®ҡд№ү
cons = ({'type': 'eq', 'fun': constraint})

# еҸҳйҮҸиҫ№з•Ңпјҡx>0, y>0
bounds = ((1e-6, None), (1e-6, None))  # з”ЁжһҒе°ҸеҖјд»ЈжӣҝдёҘж ј>0

# еҲқе§ӢеҖјпјҲйҡҸдҫҝз»ҷдёҖдёӘж»Ўи¶ізәҰжқҹзҡ„зӮ№пјү
x0 = np.array([2, 3])

# жұӮи§Ј
result = minimize(objective, x0, method='SLSQP', bounds=bounds, constraints=cons)

# иҫ“еҮәз»“жһң
print("жұӮи§ЈзҠ¶жҖҒпјҡ", result.success)
print("жңҖдјҳи§Ј x =", round(result.x[0], 4))
print("жңҖдјҳи§Ј y =", round(result.x[1], 4))
print("жңҖеӨ§еҖј xy =", round(-result.fun, 4))
```

```text

жңҖдјҳи§Ј x = 4.0
жңҖдјҳи§Ј y = 2.0
жңҖеӨ§еҖј xy = 8.0
```

## ж–№жі•дёҖпјҡеҲ©з”Ё CVXPY

йҖҡиҝҮ [CVXPY](https://pypi.org/project/cvxpy/) еә“д№ҹеҸҜд»Ҙи®Ўз®—иҝҷз§ҚйқһзәҝжҖ§и§„еҲ’й—®йўҳпјҢд»Ҙжӣҙзӣҙи§Ӯзҡ„ж–№ејҸйҖҡиҝҮиЎЁиҫҫејҸжҸҸиҝ°й—®йўҳ

```python
# pip install cvxpy
import cvxpy as cp

# е®ҡд№үеҸҳйҮҸ
x = cp.Variable(nonneg=True)
y = cp.Variable(nonneg=True)

# зӣ®ж ҮеҮҪж•°
objective = cp.Maximize(x * y)

# зәҰжқҹ
constraints = [x + 2*y == 8]

# жһ„е»әй—®йўҳ
prob = cp.Problem(objective, constraints)

# еҝ…йЎ»жҢҮе®ҡ solver=cp.SCS дё” qcp=True
prob.solve(qcp=True, solver=cp.SCS)

# иҫ“еҮәз»“жһң
print("жңҖдјҳи§Ј x =", round(x.value, 4))
print("жңҖдјҳи§Ј y =", round(y.value, 4))
print("жңҖеӨ§еҖј xy =", round(prob.value, 4))
```

иҫ“еҮә
```text
жңҖдјҳи§Ј x = 4.0
жңҖдјҳи§Ј y = 2.0
жңҖеӨ§еҖј xy = 8.0
```
