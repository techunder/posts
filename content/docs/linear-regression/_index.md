**线性回归模型**是我非常喜欢的机器学习模型，它简单、精致。

它串通了数学矩阵运算和梯度下降算法，而后两者是贯穿整个机器学习和深度学习的必备基础技能。

人工智能的核心基石是机器学习，机器学习的底层逻辑是数学原理。

我的博客不忌讳数学公式，反而会拥抱，因为**数学是全宇宙最通用、最优雅、最简约的语言**，看懂了数学，就弄清楚了世界的实质。

本文将深入拆解线性回归这一经典模型，和你一起**领略数学的简洁之美**，**体会梯度下降算法的工程实践魅力**。
