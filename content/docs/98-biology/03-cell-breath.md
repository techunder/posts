# 能量反应

细胞质膜外正离子多，细胞内外存在电势差，当蛋白质通道打开时，离子流入，产生电流，驱动动物的大脑和神经系统工作。

电子可以在光或其他辐射中获得能量（被激发），跃迁到离原子核更远的地方，甚至脱离原子核的束缚，之后如果电子坠落，就会释放出能量。化学反应中旧键断裂，新键形成，实际就是能量的转移，称为**化学能**。

> 氧化葡萄糖是放能的，身体吸入氧气氧化葡萄糖释放能量，其实和燃烧木头很相似（木头是由葡萄糖多聚体组成的纤维素），燃烧的化学方程式是：$C₆H₁₂O₆ + 6O₂ \rightarrow 6CO₂ + 6H₂O$

**放能反应**需要**活化能**的激发之后才能释放出**自由能** △G，释放出来的自由能会链式激活后续的反应，就好像燃烧需要火柴引火，葡萄糖和氧气的结合需要活化能，ATP释放能量也需要活化能激活其势能。也因为需要这一个活化能，很多易发生反应的物质才能短暂保持稳态。

**酶**可以大大降低放能反应所需要的活化能，酶的一个或多个活性部位将反应物放在合适的位置，而不是任其乱窜，因此能促进放能反应的发生。

一般来说，大分子的分解是**放能反应**（**分解代谢放能**），例如葡萄糖的氧化分解；大分子的合成则是**吸能反应**（**合成代谢吸能**），例如植物通过光合作用合成葡萄糖。

吸能反应和放能反应是成对出现的。例如，葡萄糖单体在UTP的刺激下，在酶的辅助下合成糖原。UTP在这个过程中会分解为UDP和磷酸根，释放出能量。

# 细胞呼吸

身体的各种生化反应所需要的ATP和UTP是怎么来的呢？是通过氧化葡萄糖释放出的能量去生成ATP和UTP的。宏观来看就是摄入碳水化合物，通过呼吸氧化。

食物为身体提供必要的**物质**，例如蛋白质、钙、钠、磷等，同时也是为各种生化反应提供**燃料**，例如碳水化合物。把东西吃入到体内并氧化分解，是**有机物**转化成**无机物**的过程。

氧能比碳或氢更强劲地把电子拉近，因此电子落到氧那一端时，通常会释放能量。例如，一个甲烷分子和两个氧气结合，会得到一个二氧化碳和两个水分子。
```katex
CH_4+2O_2 \rightarrow CO_2 + 2H_2O
```
有6个电子从甲烷转移到氧，这个过程会释放能量。

> [!TIP]
> 因为氧原子是以纯氧气参与反应，只需按生成物中有多少个电子被氧原子攫取了即可：每个 CO₂ 分子有2个电子被氧攫取，每个 H₂O 分子有2个电子被氧攫取。

葡萄糖氧化过程中
```katex
C_6H_{12}O_6 + 6O_2 \rightarrow 6CO_2 +6 H_2O
```
有24个电子从葡萄糖分子转移到氧，释放能量。
